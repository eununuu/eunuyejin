# 💊 안전한 의약품 복용 가이드 (Multi-page Web App)

한국의약품안전관리원의 공공데이터를 기반으로 개발된 **의약품 오남용 방지 및 병용금기(함께 먹으면 안 되는) 약물 조합 검색 서비스**입니다. 
Streamlit Cloud 환경에 단 한 개의 실행 파일(`app.py`)만으로 간편하게 배포할 수 있도록 최적화되어 있습니다.

---

## 🌟 주요 기능

### ❓ Page 1: 왜 약을 섞어 먹으면 위험할까?
* **의약품 오남용 위험성 안내**: 약물 상호작용으로 인한 체내 독성 증가 및 부작용 발생 메커니즘을 쉽게 설명합니다.
* **이상사례 시각화**: 최신 의약품 이상사례 보고 통계 데이터를 활용하여 다빈도 부작용 증상 TOP 5를 **인터랙티브 막대그래프(Plotly)**로 보여줍니다.

### 🔍 Page 2: 주요 병용금기 성분 가이드
* **성분명 기반 검색**: 사용자가 복용 중인 약물의 성분명을 리스트 선택 또는 직접 타이핑(영문/한글 지원)하여 검색할 수 있습니다.
* **병용금기 매칭 시스템**: 검색한 성분과 함께 복용해서는 안 되는 금기 성분, 해당 성분이 포함된 예시 제품명, 그리고 구체적인 **금기 사유 및 부작용 요약**을 아코디언(Expander) 형태로 직관적으로 제공합니다.
