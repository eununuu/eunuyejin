# 💊 안전한 의약품 복용 가이드 (Multi-page Web App)

한국의약품안전관리원의 공공데이터를 기반으로 개발된 **의약품 오남용 방지 및 병용금기(함께 먹으면 안 되는) 약물 조합 검색 서비스**입니다. 
Streamlit Cloud 환경에 단 한 개의 실행 파일(`app.py`)만으로 간편하게 배포할 수 있도록 최적화되어 있습니다.

---

## 🔗 사용 데이터 출처 (공공데이터포털)
본 서비스는 공공데이터포털에서 제공하는 안전성이 검증된 실시간 의약품 데이터를 활용합니다.
* [한국의약품안전관리원_연도별증상별 이상사례보고현황](https://www.data.go.kr/data/15089525/fileData.do)
* [한국의약품안전관리원_병용금기약물](https://www.data.go.kr/data/15089543/fileData.do)

---

## 👥 팀원 및 역할 소개
* **김은우** : 프로젝트 관리 및 총괄 (Project Management, UI/UX 설계, 전체 시스템 연동 및 배포)
* **유예진** : 데이터 검색 및 확인 (Data Analysis, 공공데이터 수집 및 전처리, 약물 성분/제품 매칭 로직 검증)

## 📂 프로젝트 구조 (Directory Structure)
Streamlit의 멀티 페이지 표준 구조(`pages/` 폴더 활용 방식)로 설계되어 가독성과 유지보수성이 뛰어납니다.

```text
📁 프로젝트 루트 (Repository Root)
│
├── 📄 main.py              # 메인 홈 화면 및 앱 대문
├── 📄 requirements.txt     # 설치 및 실행이 필요한 라이브러리 목록
├── 📄 README.md            # 프로젝트 소개서 (현재 파일)
│
├── 📁 pages/               # 👈 스트림릿이 자동으로 인식하는 멀티 페이지 폴더
│   ├── 📄 1_❓_왜_약을_섞어_먹으면_위험할까.py    # Page 1: 국내 부작용 통계 및 시각화
│   └── 📄 2_🔍_주요_병용금기_성분_가이드.py       # Page 2: 제품명/성분명 교차 검증 시스템
│
└── 📁 data/                # 공공데이터 CSV 파일 보관 폴더
    ├── 📄 한국의약품안전관리원_병용금기약물_20240625_2.csv
    └── 📄 한국의약품안전관리원_연도별증상별 이상사례보고현황_20241231.csv
