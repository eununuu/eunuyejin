# 💊 안전한 의약품 복용 가이드 시스템 (Medication Safety Guide App)

한국의약품안전관리원의 공공데이터를 기반으로 개발된 **의약품 오남용 방지 및 병용금기 약물 조합 검색 서비스**입니다.  
사용자가 복용 중인 약물들을 선택하면, 서로 함께 복용해서는 안 되는 위험한 약물이 있는지 실시간으로 교차 검증하고 관련 부작용 통계를 시각화하여 안내합니다.

---

## 📖 페이지 구성 및 간단 소개 (Page Overview)
본 웹 애플리케이션은 사용자의 이해를 돕는 통계 화면과 실시간 조회 시스템의 2개 페이지로 나누어 직관적인 가이드를 제공합니다.

* **Page 1. ❓ 왜 약을 섞어 먹으면 위험할까?**
  * 의약품을 잘못 혼용했을 때 체내에서 발생하는 위험성과 부작용의 메커니즘을 친절하게 설명합니다. 한국의약품안전관리원의 실제 이상사례 보고 현황 데이터를 바탕으로 가장 많이 발생하는 부작용 증상 TOP 5를 시각화 차트로 제시하여 약물 오남용에 대한 경각심을 고취합니다.
* **Page 2. 🔍 주요 병용금기 성분 가이드**
  * 환자가 현재 먹고 있는 여러 개의 약을 장바구니처럼 담아 상호 간에 같이 먹으면 안 되는 성분이 있는지 실시간으로 판별해 주는 핵심 안심 서비스입니다. 선택한 약물이 가진 전체 금기 대상 성분 리스트와 구체적인 위험 사유(부작용 종류)까지 투명하게 펼쳐볼 수 있습니다.

│

├── 📄 README.md            # 프로젝트 소개서 (현재 파일)

│   └── 📄 2_🔍_주요_병용금기_성분_가이드.py       # Page 2: 제품명/성분명 교차 검증 시스템
